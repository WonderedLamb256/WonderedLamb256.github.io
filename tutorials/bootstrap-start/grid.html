Hello, Bootstrap Containers!

Bootstrap uses the CSS flexbox feature to make advanced designs.

  • The container class is required for flexbox to work properly.
  • The row class defines a row of columns.
  • The col-sm class creates a column in the row.

Size can be increased and decreased with numbers, like col-sm-3.

Tip: Bootstrap's CSS can be overridden, and you can add new CSS classes.
